\begin{DoxyItemize}
\item This is a program written in C++ for the Raspberry Pi that is meant to vinyl-\/like funcitonality with tap cards.
\item The program uses Spotify\textquotesingle{}s API to store and play songs, with URLs being loaded onto RFID cards.
\item To play a song, we tap the corresponding card onto an RFID module attached to a Raspberry Pi while running the program.
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md3}{}\doxysubsection{Current Features}\label{md__r_e_a_d_m_e_autotoc_md3}

\begin{DoxyItemize}
\item Functional \mbox{\hyperlink{class_g_u_i}{GUI}}
\item All queued songs will play as expected
\item Cards loaded with a single song ID will be queued
\item Cards loaded with a playlist or album ID will have all its songs queued
\item Songs that are up next will are displayed on the \mbox{\hyperlink{class_g_u_i}{GUI}}
\item Users can play, pause, and skip to the next song
\item Users can add an identifier (ie. name) onto the card and the \mbox{\hyperlink{class_g_u_i}{GUI}} will display who queued which card
