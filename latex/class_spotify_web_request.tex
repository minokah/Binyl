\doxysubsection{Detailed Description}
Sends requests to the Spotify API. 

This header provides the \mbox{\hyperlink{class_spotify_web_request}{Spotify\+Web\+Request}} class to send CURL requests with appropriate headers for any Spotify API call. It is a modified version of a libcurl example. \begin{DoxyAuthor}{Author}
Brandon Luu
